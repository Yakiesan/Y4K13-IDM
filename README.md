# Y4K13-IDM
This is a web downloader, yes a web downloader. Idk man im not that good in describing but it works like IDM but for free lol.
To install it perfectly please follow the instruction below
1.What to run on your Laptop (CMD)
You need to run the REG ADD command on your laptop so that Chrome knows where to find the instructions for your app.

1.Download your .exe from GitHub.

2.Place your .exe and your nmh-manifest.json in a permanent folder (e.g., C:\Y3K14_IDM\).

3.Open CMD as Administrator.

4.Run this command (replace the path with your actual file location):
REG ADD "HKCU\Software\Google\Chrome\NativeMessagingHosts\com.y3k14.download" /ve /t REG_SZ /d "C:\Y3K14_IDM\nmh-manifest.json" /f

5.Click on the Keygen.bat 

That will make the program run smoothly!
